# durandcharlotte.github.io — Modern theme

This is a minimal, modern Jekyll theme with a left sidebar and subtle animations.

What’s included
- Responsive sidebar with profile and navigation
- Pages: Home, Publications, Presentations, Projects, CV
- CSS with modern visual style and animations
- Small JS for scroll reveal and smooth scrolling

How to use
1. Update _config.yml (site title, author, url).
2. Replace `assets/images/avatar.png` with your photo and add `assets/cv.pdf`.
3. Commit and push to GitHub. If repo is `username.github.io`, push to main branch; otherwise enable GitHub Pages for the branch and folder you prefer.
4. Preview locally with Jekyll: `bundle exec jekyll serve`.
